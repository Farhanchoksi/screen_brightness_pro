# CHANGELOG

## 1.0.0

*   Initial release of `screen_brightness_pro`.
*   Support for 4 platforms: Android, iOS, macOS, and Windows.
*   System-wide and window-level brightness control.
*   Built-in Native Wakelock (Keep Screen On) for all platforms.
*   Battery Level and Low Power Mode detection.
*   Smooth brightness transitions (interpolation).
*   Automatic brightness orientation handling.
*   Professional Control Center example UI.
